How Internet Works

The internet is a vast network of interconnected computers that use the Internet protocol suite (TCP/ICP) to communicate between networks and devices. The internet is a result of an experiment that is known as the Advanced Research Projects Agency Network more commonly referred to as the Arpanet. Continue reading

DNS also know as the Domain Name system is essentially the phonebook of the internet. DNS is a naming database in which internet domain names are located and translated into IP addresses. The domain name system maps the name people use to locate a website to the IP address that a computer uses to locate a website. Continue reading

IP stands for “internet protocol” which is a set of rules that dictate how data should be delivered over the public network (internet). An IP address is the identifier that allows information to be sent between devices on a network: they contain local information which allow devices to be accessible for communication. Continue reading

HTTP also known as Hypertext Transfer Protocol is the foundation of any data exchange on the web. HTTP is an application-layer protocol that gives users a way to interact with web resources for transmitting hypermedia documents, such as HTML between clients and servers. HTTP clients generally use Transmission Control Protocol (TCP) connections to communicate with servers Continue reading

In 1989 Tim Berners-Lee wrote a proposal for a system called the world wide web he then wrote the first web browser, server, and web page. He wrote the first specifications for URLs, HTTP, and HTML. W3C also known as the World Wide Web consortium is an organization that develops web standards. Continue reading
