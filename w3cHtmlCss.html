W3C, HTML and CSS

What is W3C and what does it do?

In 1989 Tim Berners-Lee wrote a proposal for a system called the world wide web he then wrote the first web browser, server, and web page. He wrote the first specifications for URLs, HTTP, and HTML. W3C also known as the World Wide Web consortium is an organization that develops web standards. The W3C’s primary activity is to develop protocols and guidelines that ensure long-term growth for the web. The W3C also defines key parts of what make the world wide web work.

What is HTML and what is it used for?

HTML stands for Hyper Text Markup Language. HTML Is the standard language used for creating web pages. “Hypertext” refers to the links that an HTML page may contain, and “Markup Language” refers to the way tags are used to define the page layout and elements within the page. HTML elements tell the browser how to display the content. For example, content could be structured as a set of paragraphs, a list of bullet points, or using images and data tables.

What is an example of HTML?

A user can create a paragraph by placing the enclosed text within a starting <p> and closing </p> tag. <p> Hello my name is Aishwarya Allam. I am currently pursuing my MBA with a concentration in Management Information Systems at the New Jersey Institute of Technology. I am creating this website as a part of my course work curriculum for my web systems development class. </p>

What is CSS and what is it used for?

CSS stands for Cascading Style Sheets and is used to specify your documents style for example- page layouts, colors, and fonts are all determined by CSS. CSS brings style to your web pages by interacting with HTML elements.

What is an example of CSS?

Supposed I wanted to make my paragraph above appear pink and bold to the users viewing my web page through a web browser my CSS code would look like this:

<p style= { color:pink; font-weight:bold; }
